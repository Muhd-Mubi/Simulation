Simulation Guide
Quick Start
  1. Select a queueing model (M/M/1 to M/M/4)
  2. Set simulation duration (minutes)
  3. Enter mean arrival time (minutes/customer)
  4. Enter mean service time (minutes/customer)
  5. Click "Run Simulation" to generate results
Key Insight

Utilization (ρ) = λ/(cμ) must be < 1 for stable system

Where λ = arrival rate, μ = service rate, c = number of servers

Understanding Metrics
  • Lq: Average number of customers waiting in queue
  • Wq: Average waiting time before service begins
  • W: Total time in system (waiting + service)
  • L: Average number of customers in entire system
For accurate results, run simulations for at least 480 minutes (8 hours)
