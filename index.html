M/G/c Simulation Guide
Key Concepts
  • Arrivals: Poisson process (exponential inter-arrival times)
  • Service Times: Gamma distribution with shape parameter k
  • Shape Parameter (k): Controls service time variability
    • k=1: Exponential distribution (high variance, M/M/c)
    • k>1: Lower variance (more predictable service)
    • k→∞: Near-constant service times (D)
Theoretical Formulas

M/G/1 (Single Server):

Wq = [ρ × (1/μ) × (1 + Cs²)] / [2 × (1 - ρ)]

Where Cs² = 1/k (squared coefficient of variation)

M/G/c (Multi-Server):

WqM/G/c ≈ WqM/M/c × (1 + Cs²)/2

Understanding Metrics
  • Lq: Average number of customers waiting in queue
  • Wq: Average waiting time before service begins
  • W: Total time in system (waiting + service)
  • L: Average number of customers in entire system
  • ρ: System utilization (must be < 1 for stability)
For accurate results, run simulations for at least 480 minutes (8 hours). Lower shape parameter (k) increases service time variability and queue lengths.
